# 公开平台范围与边界

本仓库的公开实现用于阵列信号处理、电磁兼容/防护、统计建模和论文复现。

- 所有信号功率相对噪声归一化；
- 不包含真实高功率源、脉冲源、链路预算或射程计算；
- 不包含具体设备易损阈值、毁伤概率标定或目标设备参数；
- 场分布模块输出归一化相干增益，不用于推断实际损伤；
- 后续防护模块聚焦接收端抗干扰与鲁棒性。

任何从归一化模型推回现实设备效应的外推都需要独立、合法、经授权的实验数据和安全审查，不能由本仓库的数值结果替代。

V0.4新增内容仅限接收端归一化抗干扰：方向置信域、宽零陷、输出SINR、白噪声增益和更新时间。任何“防护成功率”均由配置中的归一化输出SINR门限定义，不代表真实设备抗毁等级。


V0.5新增内容仍限于防御性接收处理：移动干扰源感知、方向协方差、时间戳跟踪、多扇区零陷、通道故障诊断和归一化输出SINR。动态轨迹、INR和“防护可用率”均为算法压力测试参数，不代表现实源功率、覆盖范围或设备抗毁等级。


V0.6新增内容只研究无量纲近场幅值分布、阵元幅相不确定性、空间配准抖动和归一化PA非线性。目标幅值、饱和尺度、旁区门限与“控制成功率”均为算法压力测试参数，不代表真实源功率、射程、设备易损阈值或毁伤概率。DPD模块只反演无记忆归一化曲线，不包含真实功率器件偏置、热设计或硬件驱动参数。

V0.7新增内容只研究归一化移动目标区、带时间戳的位置估计、无量纲场分布和在线控制误差。轨迹速度、时延、目标幅值、旁区p95和“动态可用率”均为算法压力测试参数，不代表现实平台跟踪能力、源功率、作用距离、目标设备阈值或物理效应概率。动态反馈使用模拟的归一化目标区平均幅值，不是现实设备状态回传。

V0.8新增内容只研究无量纲场量、累积代理剂量、统计阈值层和归一化资源分配。三档响应名称、概率、保护区风险和“任务可用率”均为算法比较指标，不对应具体器件失效模式、现实毁伤概率、源功率、作用距离或目标平台参数。EA-Duty仅在0到1之间选择抽象数值权重，不提供现实硬件脉冲、源驱动或功率调度参数。
