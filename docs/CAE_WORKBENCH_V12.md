# HPM-CAE Workbench V1.2 技术说明

## 1. 版本目标

V1.2 解决 V1.1 双目标示例中“目标区均匀性尚可，但局部区外峰值和保护区约束容易失守”的问题。升级重点不是再换一种启发式算法，而是把场景对象的要求写成可检查、可视化、可扫参的约束体系。

平台仍采用波长尺度几何、标量 Green 矩阵、归一化复场和无量纲工程判据。所有结果用于算法研究与软件验收，不等同于全波电磁仿真或实物效应结论。

## 2. 对象级场景参数

目标对象 `TargetRegionSpec` 新增：

- `priority`：目标在联合目标函数中的相对权重；
- `tolerance_percent`：对象级幅度允许误差；
- `amplitude_scale`：相对全局目标幅度的独立设定值；
- `guard_scale`：目标边界与普通区外约束之间的过渡带。

保护对象 `ProtectedZoneSpec` 新增：

- `priority`：保护区风险项权重；
- `max_amplitude_scale`：相对全局目标参考幅度的独立上限。

普通区外还设置独立的全局峰值判据 `outside_peak_limit_db`。优化中的 `outside_hinge_amplitude` 是软铰链阈值，工程验收中的峰值上限则由 `outside_peak_limit_db` 明确定义，两者不再混用。

## 3. Constrained-MO-PGMS

对第 \(k\) 个目标对象，设归一化幅度误差为

\[
e_k(\mathbf w)=\frac{|\mathbf A_k\mathbf w|-\mathbf d_k}{\mathbf d_k}.
\]

名义目标项采用对象优先级加权的均方误差，并加入平滑最大值形式的公平性项，防止大目标吞噬小目标：

\[
J_{\mathrm{target}}
=\sum_k \alpha_k\,\mathbb E[e_k^2]
+\lambda_f\,\operatorname{smax}_k\big(\mathbb E[e_k^2]\big).
\]

普通区外和保护区分别使用超限铰链损失：

\[
J_{\mathrm{outside}}=\lambda_o\,\mathbb E[(|\mathbf B\mathbf w|-\tau_o)_+^2],
\]

\[
J_{\mathrm{protected}}=\lambda_p\sum_j\beta_j\,
\mathbb E[(|\mathbf C_j\mathbf w|-\tau_j)_+^2].
\]

尾部峰值项只对区外幅度最高的一部分样本加权，抑制“均值好看但局部漏光”的情况。总目标还包含小幅能量正则化；每次 Adam 更新后将复权值投影到 RMS 和单阵元峰值约束集合。

鲁棒模式同时对阵元增益误差、相位误差和观察面配准扰动采样，多场景共享同一组权值。

## 4. 对象级评价

每次静态求解输出：

- 目标对象：RMSE、容差覆盖率、P95 偏差、均值、峰值和通过状态；
- 保护对象：P95、峰值、上限、超限 dB 和通过状态；
- 全局：总体 RMSE、最差目标 RMSE、最低覆盖率、公平性差距、区外峰值、保护区最坏超限、约束通过率和联合判据。

二维约束裕量图采用统一符号：正值（蓝色）表示满足，负值（红色）表示超限。目标区显示相对对象容差的剩余裕量，保护区显示相对独立幅度上限的裕量，普通区外显示相对全局峰值上限的点级裕量。

## 5. Pareto 设计空间

Pareto 模块同时缩放普通区外与保护区风险惩罚，逐点运行真实求解器。横轴为最差目标 RMSE，纵轴为最坏风险超限：

\[
r=\max\{P_{\mathrm{out}}-P_{\mathrm{limit}},\;v_{\mathrm{protected}}\}.
\]

`r ≤ 0` 表示所有风险上限均满足。模块自动标记非支配点，并基于归一化到理想点的距离、辅以小幅能量效率惩罚，选择一个推荐折中方案。推荐是算法辅助，不替代研究者对场景约束的判断。

## 6. UI 工作流

1. 在“场景 / 对象”页添加或编辑目标区与保护区；
2. 为每个目标设置优先级、容差和幅度倍率，为每个保护区设置优先级与上限；
3. 在左侧“空间控场”配置惩罚权重、尾部比例、误差场景和功放模型；
4. 运行“静态约束控场”，查看场图、约束裕量、对象指标和分项收敛；
5. 在“Pareto 设计空间”运行快速或完整扫描；
6. 导出自包含 HTML、CSV、NPZ、YAML 和完整 ZIP；
7. 使用任务队列做多次重复或参数敏感性分析。

## 7. 默认双目标结果

默认 V1.2 工程在 320 次迭代、5 个鲁棒场景下达到：总体 RMSE 8.006%，最差目标 RMSE 8.012%，最低目标覆盖率 71.556%，区外峰值 −2.202 dB，最坏保护区超限 −1.358 dB，对象约束通过率 100%，联合判据通过。

结果依赖当前归一化模型、网格和随机种子。论文使用前应增加 Monte Carlo 重复、参数外测试、模型失配和基准消融。

## 8. 后续接口

V1.3 建议优先建设插件式“场求解后端”接口，使同一 UI 可切换：标量 Green 代理、射线/镜像多径代理、孔缝—腔体降阶模型以及外部数据导入。这样 UI 和实验管理层保持稳定，物理求解内核可逐级升级。
