# 用户使用手册

## 1. 打开工程

启动后默认进入“城市多对象动态覆盖”示例。顶部下拉菜单可切换到静态调查或空白工程。

## 2. 理解界面

- 左侧“场景树”：只显示物理对象和探针；
- 左侧“对象库”：添加阵列平台、运动对象、建筑、接收设备和场切片；
- 中央“三维场景”：查看对象、轨迹和场结果；
- 右侧“属性检查器”：修改选中对象的位置和旋转；
- 底部“时间线”：播放动态仿真；
- 底部“接收曲线”：查看每个接收实体随时间的归一化场幅度。

## 3. 运行示例

点击右上角“运行任务”。完成后：

1. 拖动时间线查看不同帧；
2. 点击播放按钮连续回放；
3. 点击三维对象或场景树对象查看属性；
4. 打开“接收曲线”比较不同实体。

## 4. 修改场景

在对象库添加任意对象。选中对象后，可在右侧修改位置和旋转。修改后需要重新运行任务。

## 5. 为什么没有“目标区1”

运动对象通过 `role:trackable` 自动成为动态任务参与对象。空间场切片是分析探针，不是目标本身。这样同一个任务可以适用于任意数量实体。

## 6. 保存

点击“保存工程”下载 `.hpmdt` 文件。该文件包含场景、任务和已运行结果。
