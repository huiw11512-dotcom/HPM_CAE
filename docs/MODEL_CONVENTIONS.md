# 统一建模与接口约定 V0.2

## 1. 坐标系

- 阵列位于 `x-y` 平面，阵心在原点，阵面法向为 `+z`；
- `theta` 为相对阵面法向的偏转角，`theta=0°` 表示宽侧方向；
- `phi` 为方位角，从 `+x` 朝 `+y` 旋转；
- 远场方向余弦：`u=sin(theta)cos(phi)`，`v=sin(theta)sin(phi)`。

## 2. 数组形状

| 对象 | 形状 | 说明 |
|---|---:|---|
| 阵元坐标 | `(M, 3)` | 米制坐标 |
| 阵列快拍 | `(M, L)` | M个阵元，L个快拍 |
| 导向矢量 | `(M,)` | 单一方向 |
| 导向矩阵 | `(M, G)` | G个扫描方向 |
| 阵列数据立方体 | `(Nx, Ny, L)` | 空间平滑内部表示 |
| 子阵协方差 | `(Ms, Ms)` | `Ms=Lx×Ly` |
| 场采样点 | `(P, 3)` | P个空间点 |
| 场图/概率图 | `(Nz, Nx)` | `x-z`平面 |

阵元展平顺序为 `x` 维在外、`y` 维在内，与 `reshape(nx, ny, snapshots)` 一致。

## 3. 相位约定

- 接收导向矢量：`a = exp(+j k r_n · s)`；
- 接收响应：`|wᴴa|`；
- 发射标量传播核：`exp(-j k R_n)/R_n`；
- 相位共轭聚焦权：`q_n ∝ exp(+j k R_n)`；
- 相干多径超额相位：路径复系数乘 `exp(+j phase)`。

## 4. 窄带源与相干多径

独立源基线：每个方向使用独立复高斯波形。相干多径模型：同一 `CoherentEmitter` 内所有 `CoherentPath` 共用同一个时间波形，只具有不同导向矢量、相对功率和超额相位。

功率参数均相对噪声功率以dB给出；不映射到真实源输出或绝对场强。

## 5. 空间平滑

矩形阵列数据先恢复为 `(Nx, Ny, L)`。所有连续 `Lx×Ly` 子阵按相同展平顺序计算协方差并平均。前后向平均使用 `Ms×Ms` 反单位矩阵：

```text
R_FB = 0.5 × (R_SS + J R_SS* J)
```

子阵对象重新居中只会给每个源引入公共相位，不改变协方差或 MUSIC 谱。

## 6. MUSIC 与角度误差

MUSIC扫描结果归一化到 `[0,1]`。快速实现使用：

```text
||a||² - ||E_sᴴa||²
```

代替显式噪声子空间投影，两者由完备正交子空间恒等式保证等价。

方向误差采用三维单位方向向量的球面夹角，而不是简单的 `theta/phi` 欧氏距离。多峰与多真值之间采用最小总代价匹配。

## 7. 当前物理层级

V0.2 包含：

1. 窄带独立源与相干多径阵列观测；
2. 阵元随机幅相误差；
3. 二维MUSIC与矩形阵列空间平滑；
4. 标量格林函数空间叠加；
5. 方向图、相干增益和归一化统计响应模型。

尚不包含：阵元互耦、极化、宽带色散、孔缝/腔体模态、真实功率链路、器件标定阈值和全波边界条件。这些内容应作为独立模型插件接入，不能混入一个不可解释的“万能信道函数”。

## 8. 近场可视化量

`near_field_coherence()` 输出：

```text
|gq|² / (||g||² ||q||²)
```

它是范围归一化的相干增益，取值在 `[0,1]`，用于观察聚焦与旁区结构；它不是绝对场强。

## 9. 效应输出

公开版本采用无量纲应力与对数正态阈值分布：

```text
归一化强度 → 应力指数 → 响应概率
```

输出只用于算法比较、不确定性传播和敏感性分析，不能直接解释为某具体设备的真实失效概率。

## V0.4接收防护补充约定

- 训练协方差中的干扰方向为感知中心；评估方向可在权值生成后漂移；
- 通道幅相误差乘在信号阵列流形上，接收机白噪声保持独立；
- 输出SINR使用解析独立评估式，不复用训练快拍估计性能；
- `actual_null_db`相对于同一权值下的真实目标方向响应归一化；
- `protection_rate`仅表示输出SINR达到配置门限的比例。


## V0.6近场区域调控约定

- 控制平面使用标量Green函数 `exp(-j k R)/R`，以单位阵元RMS的目标中心点聚焦场幅值归一化；
- 目标区、保护间隔和旁区互斥且覆盖整个采样平面；保护间隔不进入旁区惩罚；
- 优化变量首先解释为期望PA输出权值，DPD再生成有界驱动；
- 阵元增益/相位误差作用于PA输出之后，配准抖动作用于场采样坐标；
- 所有“成功”均由配置中的无量纲联合门限定义，禁止解释成设备效应或毁伤结论。
