# 第三方开源组件说明

HPM 数字化电磁算法 CAE V1.4 的本地 Web 工作台使用以下开源前端组件。相关文件已随项目本地化，不依赖公共 CDN。

| 组件 | 本项目用途 | 版本/来源 | 许可证 |
|---|---|---|---|
| Bootstrap | 官方 Dashboard 示例结构、网格、表单、卡片、导航与响应式布局 | Bootstrap 5.1.1 | MIT |
| Bootstrap Icons | 侧栏与操作按钮图标 | MNE-Python 随附的嵌入字体版本 | MIT |
| Plotly.js | 本地交互式二维/三维图形 | Plotly.js 3.3.1 | MIT |
| Three.js | V2.0B 本地三维 CAE 视口 | Three.js 0.166.1 | MIT |

本项目的界面布局沿用 Bootstrap 官方 Dashboard 示例的通用结构，并在其上增加中文 CAE 页面与业务交互；没有把自绘管理后台皮肤冒充成第三方模板。

许可证全文见：

- `licenses/BOOTSTRAP_LICENSE.txt`
- `licenses/BOOTSTRAP_ICONS_LICENSE.txt`
- `licenses/PLOTLY_LICENSE.txt`
