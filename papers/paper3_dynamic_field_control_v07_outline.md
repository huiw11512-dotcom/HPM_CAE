# 论文三升级版工作骨架：动态预测协方差与反馈区域调控

## 暂定题目

**Predictive Covariance-and-Feedback Region Control for a Moving Normalized Near-Field Zone under Sensing Latency and Array Uncertainty**

中文工作题目：**感知滞后与阵列不确定条件下移动近场区域的预测协方差—反馈调控方法**

## 核心科学问题

延迟、噪声和机动导致目标区中心在执行时刻存在偏差；仅以点估计更新权值会在机动段出现区域漏照，仅扩大目标区又会恶化旁区暴露。本文研究如何将时间戳预测、二维位置协方差、阵元幅相场景和归一化场反馈统一映射到低时延区域控制问题。

## 候选贡献

1. 建立带采集时间戳、处理滞后、执行滞后和反馈滞后的动态空间场评价协议；
2. 提出协方差主轴Sigma中心与置信包络分离机制，使目标场景和旁区约束保持物理语义；
3. 提出PCF-RLS：场景平均区域最小二乘、少量交替相位更新、硬件场景集、DPD与有界反馈；
4. 给出时延、测量噪声、机动强度和相位误差下的配对Monte Carlo、消融、失败边界和更新时间；
5. 明确报告RMSE—覆盖率—旁区p95—运行时间折中，不宣称所有指标全面领先。

## 当前快速结果

代表场景PCF-RLS：RMSE 9.84%、覆盖率66.2%、动态可用率56.3%、更新中位数14.56 ms。Predictive-RLS可用率59.4%，因此文章主张应限定为“提高时均区域精度和覆盖率”，不能写成“所有联合指标最优”。

快速Monte Carlo每点3次，基准工况的配对检验不显著；正式论文至少运行60次/点，并固定随机种子、记录环境和报告效应量。
