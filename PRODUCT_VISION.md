# 产品愿景

HPM-DT Studio 的目标不是复刻一个缩小版 CST，也不是继续堆叠算法、评分和论文报表。

它要解决的问题是：

> 在包含多个物理对象、多个阵列系统、复杂环境和动态运动的场景中，配置一项可复用的电磁任务，运行系统级数值仿真，并直观观察时变场分布、传播路径、阵列响应和对象级结果。

## 用户看到的主流程

新建工程 → 建场景 → 配系统 → 建任务 → 运行 → 看结果 → 保存

## 平台差异化

全波软件擅长结构级电磁求解；HPM-DT Studio 聚焦系统级、任务级和时间动态的组合仿真。

## 不做什么

- 不把可信度评分作为首页；
- 不把论文报告作为产品主流程；
- 不使用固定“目标1/目标2”编号决定算法行为；
- 不把数学区域伪装成现实对象；
- 不将归一化代理量写成现实毁伤结论。
