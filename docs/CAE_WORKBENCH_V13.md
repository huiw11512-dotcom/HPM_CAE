# HPM 数字化电磁算法 CAE V1.3 技术说明

## 1. 版本目标

V1.3 将 V1.2 中写死在静态求解器里的自由空间标量 Green 矩阵抽象为可注册后端，并引入材料、反射面、孔缝和腔体四类环境对象。算法层只依赖线性传播矩阵，因此无需重写多目标赋形、鲁棒优化、功放或评价代码。

## 2. 统一接口

后端核心接口为：

\[
\mathbf H = \mathcal B(\mathcal A,\,\mathbf R,\,\mathcal E),
\]

其中：

- \(\mathcal A\) 为阵列几何；
- \(\mathbf R\) 为采样点坐标；
- \(\mathcal E\) 为材料和环境对象；
- \(\mathbf H\in\mathbb C^{P\times M}\) 为采样点到阵元的线性传播矩阵。

场计算统一写为：

\[
\mathbf e = \mathbf H\mathbf w.
\]

## 3. 内置后端

### 3.1 自由空间标量格林

\[
H_{pm}=\frac{\exp(-jkR_{pm})}{R_{pm}s_{\mathrm{ref}}}g_m.
\]

它用于回归校验、算法基线和无环境快速求解。

### 3.2 一阶镜像射线

对每个启用反射面构造阵元镜像位置，并叠加一阶反射路径。材料参数只作为归一化反射幅度、相位和粗糙度代理，不等价于严格 Fresnel 全波边界。

### 3.3 孔缝—腔体降阶模型

孔缝中心先接收阵列入射场，再通过有限矩形模态基映射到腔体内部采样点。耦合系数、品质因数、泄漏系数和模态数量均为归一化研究参数。

### 3.4 混合场景

混合后端将直达、镜像反射和腔体降阶分量线性叠加，用于验证优化器对传播模型变化和环境不确定性的适应性。

## 4. 鲁棒场景生成

每个鲁棒场景统一抽样：

- 阵元增益误差；
- 阵元相位误差；
- 目标/观察面注册偏差；
- 当前工程选定的传播后端。

同一场景中的目标区、普通区外和保护区共享同一个误差向量和注册偏移，避免对象之间使用不一致的随机扰动。

## 5. 工程对象

- `MaterialSpec`：材料标识、相对介电常数、损耗角正切、反射幅相和粗糙度代理；
- `ReflectingPlaneSpec`：反射面法向轴、坐标和材料引用；
- `ApertureSpec`：孔缝中心、等效半径、耦合系数和腔体引用；
- `CavitySpec`：中心、三维尺寸、品质因数、泄漏系数和各轴模态数量；
- `PropagationSpec`：当前后端、各分量系数和最大模态数。

## 6. 中文 UI

V1.3 使用开源 Gradio Ocean 主题，不再维护自绘后台 CSS。左侧为原生 `Sidebar` 参数区，主视区使用原生 `Tabs`，场景对象使用 `Dataframe`，三维几何和结果使用 Plotly。

## 7. 模型边界

本版属于阵列算法 CAE 和降阶传播研究工具，不是全波求解器。所有结果均为波长尺度、归一化复场和无量纲代理评价，不能用于推断真实源功率、设备阈值、毁伤概率或作用距离。
