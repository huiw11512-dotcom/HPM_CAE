# HPM-DT 项目愿景

HPM-DT（High-Power Microwave Digital Twin）的长期目标，是持续演进为一个面向高功率微波系统、相控阵、传播效应、数字孪生和科研论文生产的全中文开源 CAE 平台。

本项目不是算法仓库、单个论文项目或若干 Python 脚本；它的目标形态是科研级软件系统，组合能力对标 MATLAB Phased Array Toolbox、CST 后处理平台、COMSOL App、数字孪生平台和论文流水线。

## 平台定位

- 面向科研、论文、数字孪生和算法验证。
- 长期保持归一化研究平台定位。
- 不替代 CST、HFSS 或 COMSOL。
- 不输出真实毁伤概率、真实作用距离或真实器件失效预测。

## 最终工作流

新建工程 -> 建立场景 -> 配置阵列 -> 配置目标与保护区 -> 运行求解 -> 自动验证 -> 自动统计 -> 自动生成图表 -> 自动生成论文。

详细 North Star 见 [docs/HPM_DT_NORTH_STAR.md](docs/HPM_DT_NORTH_STAR.md)。
