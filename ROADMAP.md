# HPM-DT 路线图

本文件是根目录路线图，细化版本见 [docs/HPM_DT_ROADMAP.md](docs/HPM_DT_ROADMAP.md)。

## V2.x

| 版本 | 目标 | 当前状态 |
|---|---|---|
| V2.0A | 可信度验证体系：解析验证、文献复现入口、不确定度、敏感性、可信度评分 | 已完成核心链路 |
| V2.0B | 三维 CAE 编辑器：Three.js、对象树、三维场景、属性面板、求解联动 | 预览接入中 |
| V2.0C | 任务级仿真框架：任务模板、目标运动时间线、归一化风险代理、任务产物；插件系统作为扩展基础设施 | 最小闭环已接入 |
| V2.0D | 论文自动生产线：实验、统计、图表、LaTeX、论文草稿 | 预览接入中 |

## V3.0

接入真实数据，支持 CST/HFSS 导出包、测量数据、CSV、Touchstone、NPZ、HDF5，并形成数据血缘、坐标/单位、不确定度和标定桥接审计。

## V4.0

形成 HPM 数字孪生平台，可支撑软件著作权、SCI 论文体系、博士课题平台、项目申报材料和实验室公共平台。
